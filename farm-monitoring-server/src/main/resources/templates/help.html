Реєстрація в системі

Для початку роботи з системою, адміністратор повинен створити обліковий запис для користувача:

  1. Зверніться до системного адміністратора для отримання облікового запису
  2. Увійдіть в систему за допомогою отриманих облікових даних
  3. Змініть тимчасовий пароль на свій власний
Пароль повинен містити мінімум 8 символів, включаючи літери, цифри та спеціальні символи.
Інтерфейс панелі керування

Панель керування надає доступ до всіх основних функцій системи. Основні розділи доступні через бічне меню:

  • Дашборд - загальний огляд стану ферм
  • Датчики - управління датчиками
  • Поля - перегляд і налаштування полів
  • Аналітика - детальний аналіз зібраних даних

Налаштування профілю користувача

Після першого входу в систему рекомендується налаштувати свій профіль:

  • Натисніть на ім'я користувача у верхньому правому куті
  • Виберіть "Налаштування профілю"
  • Заповніть інформацію та налаштуйте сповіщення
Відео-інструкція: Початок роботи

Перегляньте відео-інструкцію, щоб швидко ознайомитися з основними функціями системи

Часті питання

Для зміни мови інтерфейсу перейдіть у розділ "Налаштування профілю" і виберіть бажану мову зі списку доступних.

На сторінці входу натисніть "Забули пароль?" і слідуйте інструкціям. Вам потрібно буде ввести електронну пошту, прив'язану до вашого облікового запису.

Типи датчиків

В системі використовуються різні типи датчиків для моніторингу умов на фермі:

Температурні датчики

Вимірюють температуру повітря або ґрунту в градусах Цельсія. Ідеально підходять для контролю умов зростання рослин.

Робочий діапазон: від -40°C до +85°C
Датчики вологості

Вимірюють вологість ґрунту у відсотках. Допомагають оптимізувати полив і запобігти пересиханню або перезволоженню.

Робочий діапазон: від 0% до 100%

Інтерпретація даних датчиків

Стан батареї датчика

Система відображає стан заряду батареї для кожного датчика:

85%
Хороший 60-100%
45%
Середній 30-59%
15%
Критичний 0-29%
Статус підключення

Статус підключення датчика відображається за допомогою кольорових індикаторів:

Активний Датчик передає дані регулярно
Попередження Датчик пропускає деякі передачі даних
Неактивний Датчик не передає дані більше 24 годин

Додавання нового датчика

  1. Перейдіть на сторінку "Датчики" в головному меню
  2. Натисніть кнопку "Додати новий датчик"
  3. Заповніть необхідні поля у формі:
    • Серійний номер датчика (вказаний на пристрої)
    • Тип датчика (температура/вологість)
    • Назва (для зручної ідентифікації)
    • Поле, до якого належить датчик
    • Координати розташування (за необхідності)
  4. Натисніть "Зберегти" для додавання датчика в систему
Важливо: Переконайтеся, що серійний номер датчика введено правильно. Це унікальний ідентифікатор, який не можна буде змінити після активації.
Бездротове підключення

Після додавання датчика в систему, необхідно активувати його бездротове підключення:

  1. Вставте батареї в датчик
  2. Натисніть і утримуйте кнопку перезавантаження на 5 секунд
  3. Дочекайтеся, поки індикатор почне блимати зеленим
  4. Датчик автоматично підключиться до системи протягом 5 хвилин

Додавання нового поля

Для додавання нового поля в систему:

  1. Перейдіть на сторінку "Поля" в головному меню
  2. Натисніть кнопку "Додати нове поле"
  3. Заповніть інформацію про поле:
    • Назва поля
    • Ферма, до якої належить поле
    • Площа (в гектарах)
    • Тип ґрунту
    • Поточна культура
    • Координати поля (за необхідності)
  4. Натисніть "Зберегти"
Визначення меж поля

Система дозволяє визначити точні межі поля на карті:

  1. На сторінці поля натисніть "Редагувати межі"
  2. Використовуйте інструменти малювання для окреслення меж поля на карті
  3. Натисніть "Зберегти межі"
Визначення точних меж покращує візуалізацію даних та дозволяє створювати теплові карти.

Оптимальне розміщення датчиків

Для ефективного моніторингу всього поля важливо правильно розмістити датчики:

Рекомендації з розміщення
  • Рівномірне покриття - розміщуйте датчики на рівних відстанях один від одного
  • Врахування рельєфу - розміщуйте додаткові датчики в низинах і на підвищеннях
  • Враховуйте особливості поля - ділянки з різними типами ґрунту потребують окремих датчиків
Щільність розміщення датчиків

Рекомендована кількість датчиків залежить від площі поля та однорідності ґрунту:

Площа поля Мінімальна кількість датчиків Оптимальна кількість датчиків
до 10 га 3-5 5-10
10-50 га 5-10 10-20
50-100 га 10-20 20-40
більше 100 га 20+ 40+

Робота з аналітичними даними

Система надає потужні інструменти для аналізу даних, зібраних з датчиків:

Основні функції аналітики
  • Фільтрація даних - за типом датчиків, періодом часу, полями та фермами
  • Порівняльний аналіз - порівняння показників різних полів або періодів
  • Тренди та прогнози - аналіз динаміки показників та прогнозування майбутніх значень
  • Теплові карти - візуалізація розподілу показників по всій площі поля
  • Експорт даних - збереження даних у форматах Excel, PDF, CSV
Інтерпретація графіків

Система використовує різні типи графіків для відображення даних:

  • Лінійні графіки - для відображення трендів і динаміки показників у часі
  • Стовпчасті діаграми - для порівняння значень різних датчиків або полів
  • Кругові діаграми - для відображення співвідношень між різними категоріями даних
  • Гістограми - для аналізу розподілу значень

Створення звітів

Система дозволяє створювати детальні звіти для подальшого аналізу та прийняття рішень:

Типи звітів
  • Щоденні звіти - містять інформацію про показники за останні 24 години
  • Тижневі звіти - надають аналіз трендів за тиждень
  • Місячні звіти - комплексний аналіз показників за місяць
  • Звіти за періодом - дозволяють обрати будь-який період для аналізу
  • Порівняльні звіти - порівняння показників різних полів або періодів
Як створити звіт
  1. Перейдіть на сторінку "Аналітика" в головному меню
  2. Налаштуйте фільтри для відображення потрібних даних:
    • Виберіть період аналізу
    • Виберіть ферму та поле (за необхідності)
    • Виберіть тип даних (температура, вологість або всі)
  3. Натисніть "Оновити дані" для застосування фільтрів
  4. Натисніть "Експорт" і виберіть потрібний формат (Excel, PDF, CSV)
Налаштування регулярних звітів

Система дозволяє налаштувати автоматичну генерацію та розсилку звітів:

  1. На сторінці "Аналітика" натисніть "Налаштування звітів"
  2. Натисніть "Додати новий регулярний звіт"
  3. Виберіть тип звіту та періодичність (щодня, щотижня, щомісяця)
  4. Налаштуйте фільтри даних для звіту
  5. Вкажіть адреси електронної пошти для розсилки
  6. Натисніть "Зберегти"
Звіти будуть автоматично генеруватися та надсилатися на вказані адреси електронної пошти.
